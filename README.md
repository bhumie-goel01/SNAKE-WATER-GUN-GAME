# SNAKE-WATER-GUN-GAME
An interactive Snake Water Gun game built from my original game logic, featuring a modern interface and smooth user experience.

# ABOUT THE PROJECT
Snake Water Gun is a Python-based game where I wrote the complete game logic using Python. The project was created to strengthen my understanding of core programming concepts such as loops, conditionals, dictionaries, and problem-solving techniques.
As I progressed in my learning journey, I wanted to transform the game into a more engaging and user-friendly experience. To achieve this, I took help of AI-assisted tools to build a modern web interface using HTML, CSS, and JavaScript. Rather than simply using generated code, I reviewed, tested, debugged, and customized the implementation to ensure it worked correctly and aligned with my vision.
This project reflects both my programming fundamentals and my ability to effectively use modern AI tools during the software development process.

# FEATURES
* Play Snake Water Gun game against a computer opponent that makes random choices.
* Track your wins, losses, draws, and overall performance.
* View your current winning streak and best streak record.
* Built-in rule book explaining game rules and available keyboard shortcuts.
* Achievement system with notifications for reaching different milestones.
* Save your progress and continue from where you left off.
* Match history section to review recent game results.
* Interactive animations and visual effects for a more engaging experience.
* Responsive user interface that works across different screen sizes.
* Personalized gameplay with custom player names.
* Modern dark and light theme support.
* Simple and beginner-friendly design suitable for players of all ages.
